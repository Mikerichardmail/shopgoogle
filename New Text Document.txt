SORTING LOGIC

1. Start with full master store list
2. Move tab priority stores to top
3. If category selected:
   Move category boost stores below tab priority
4. Keep remaining stores in original order
5. No store should be removed

---

DYNAMIC TOP STORES SECTION

Top stores section should always show:

- First 6–8 stores from sorted list
- Automatically updates when tab or category changes
